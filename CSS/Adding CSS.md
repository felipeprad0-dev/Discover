# Adding CSS 

## inline 

    * `Style` attribute  

    ```HTML 
    <h1 style="color: red;">
        Title
            <strong style="">Second Title</strong>
    </h1>
    ```


## <style>

    * HTML tag that will contain the CSS 

    ```HTML 
    <style>
        h1 {
            color: blue;
        }

        strong {
            color: red;
        }
    </style>
    ```


## <link>

    * External .css file

    ```HTML
        <link rel="stylesheet" href="style.css">
    ```

    * External .css file

    * @import inside the .css file
