# Tipos de dados 

    ECMAScript é o orgão que padroniza o JavaScript 
    Conforme o ECMAScript standard, há 9 tipos de dados: 

## Data Types 

    - Primitives / Primitive value 
    - Structural 
    - Scructural Primitive 


## Primivites 

    * Não é um objeto
    * O valor é imutável

    - String
    - Number 
    - Boolean
    - Undefined
    - Symbol 
    - BigInt


## Structural

    * Objeto
        - Array 
        - Map
        - Set
        - Date
        - ...
    
    * Function 


## Structural Root Primitive

    - Null
