# String Operator

_Use ` + ` to concatenations. It will return the joining of two or more strings._


## Examples 

```JS
console.log('alpha' + 'bet')    ----->    'alphabet'   
```

```JS
let alpha = 'alpha'

console.log(alpha + 'bet')    ----->    'alphabet'
```

```JS
let alpha = 'alpha'

console.log(alpha + 325)    ----->    'alpha325'
```
