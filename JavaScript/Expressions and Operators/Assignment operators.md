# Assignment Operators

let x;

## Assignment 

` = `

```JS
x = 1
console.log(x)      ----->      1
```


## Addition assignment

` += `

```JS
// x += 2   ----->   x = x + 2

x += 2
console.log(x)      ----->      3
```


## Subtraction assignment 

` -= `

```JS
// x -= 1   ----->   x = x - 1

x += 1
console.log(x)      ----->      2
```


## Multiplication assignment 

` *= `

```JS
// x *= 2   ----->   x = x * 2

x *= 2
console.log(x)      ----->      4
```


## Division assignment 

` /= `  

```JS
// x /= 2   ----->   x = x / 2

x /= 2
console.log(x)      ----->      2
```


## Remainder assignment 

` %= `

```JS
// x %= 2   ----->   x = x % 2
x %= 2
console.log(x)      ----->      0
```


## Exponential assignment

` **= `

```JS

let x = 2;

// x **= 2   ----->   x = x ** 2
x **= 2
console.log(x)      ----->      4
```
